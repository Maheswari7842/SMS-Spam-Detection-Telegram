# SMS Spam Detection via Telegram 🤖💬

This project uses **Machine Learning** to detect SMS spam messages and integrates the result with a **Telegram Bot** for real-time interaction.

## 📌 Project Highlights
- 💡 Detects spam vs. ham SMS using a trained ML model.
- 🤖 Telegram bot integration for easy message checking.
- 📊 Presentation included for academic or demo purposes.

## 📂 Project Structure
- `bot.py` – Main Telegram bot logic
- `spam_detector.py` – ML model for spam detection
- `dataset/spam.csv` – Training dataset
- `presentation/SMS_Spam_Telegram_Presentation.pptx` – Project overview slides
- `requirements.txt` – All required libraries

## ⚙️ Installation
1. Clone the repo:
   ```bash
   git clone https://github.com/MAheswari7842/sms-spam-detection-telegram.git
   
   cd sms-spam-detection-telegram
**Install dependencies:**
pip install -r requirements.txt


**Run the bot:**
python bot.py

**How It Works**
User sends a message to the Telegram bot.
The message is passed to the ML model.
The bot responds whether it’s “Spam” or “Ham”.
